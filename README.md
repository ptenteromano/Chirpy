# Chirpy

Using Go's http library to serve static files and assets

To clear the local database and run in debug mode, run the following command:

```bash
go build -o out && ./out --debug
```


## Whats included
- HTTP using the STD
- Simple storage using a `dataabse.json` file
- Users, Chirps
- `jwt` authentication
